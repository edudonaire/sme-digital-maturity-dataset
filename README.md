# SME Digital Maturity Dataset (Synthetic)

This repository contains a synthetic dataset designed to model the digital maturity of Small and Medium Enterprises (SMEs).  
It includes indicators related to:

- Data governance maturity  
- Process automation levels  
- BI & Analytics readiness  
- Data literacy  
- Infrastructure score  
- Overall maturity index  

The dataset was generated to be:
- **Realistic** (based on SME digital transformation frameworks)  
- **Statistically coherent**  
- **Machine-learning friendly**  
- **Free for public research and experimentation**
